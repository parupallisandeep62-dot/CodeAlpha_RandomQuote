# CodeAlpha_RandomQuoteGenerator

## Random Quote Generator App - CodeAlpha Internship Task 2

Android application built using **AIDE on Mobile** for CodeAlpha Android Development Internship.

### ✨ Features
- Display random motivational/inspirational quotes
- Shows quote text with author name clearly
- "New Quote" button to generate next quote
- Clean and simple Material Design UI
- Lightweight and fast performance

### 🛠️ Tech Stack
- **Language:** Java
- **IDE:** AIDE - Android IDE
- **Platform:** Android
- **Min SDK:** Android 5.0 (Lollipop)

### 📱 Screenshots
App shows quote text and author name on main screen with a button to fetch new quotes.
